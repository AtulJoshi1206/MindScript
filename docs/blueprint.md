# **App Name**: MindScript

## Core Features:

- User Authentication: Secure user authentication using NextAuth.js with Google OAuth and JWT sessions.
- Daily Check-in via Journaling: Authenticated users can perform daily check-ins via journaling (text input).
- Daily Check-in via MCQs: Authenticated users can perform daily check-ins via multiple-choice questions.
- Depression Score Prediction: Gemini tool predicts a depression score and level based on journal entries and MCQ responses.
- Dynamic Dashboard Visualization: Visualize daily check-in results and depression scores on a dynamic dashboard with a Mental Curve Graph.
- AI-Powered Empathetic Feedback: Gemini agent provides empathetic feedback, actionable precautions, and conversational support based on the user's mental state and journal entries.
- Data Storage: Stores user data, check-in history, and scores in a local SQLite database file.

## Style Guidelines:

- Primary color: Deep Calming Teal (#006D77), evoking tranquility and focus, mirroring the application's purpose in promoting mental wellness. Its depth provides a sense of stability, very important for mental health apps.
- Background color: Desaturated Teal (#E0F2F7) for a soft, airy background that complements the primary teal.
- Accent color: Soft Sky Blue (#83C5BE) as a secondary accent for interactive elements, to encourage a feeling of calm.
- Headline font: 'Belleza' sans-serif for headlines and short chunks of text, giving personality to the design.
- Body font: 'Alegreya' serif for body text; for readability and a friendlier experience.
- Use minimalist, calming icons with rounded corners.
- Rounded corners (border-radius: 12px+) and soft shadows for a safe, non-judgmental feel.
- Use subtle, slow transitions and animations to avoid jarring the user.
